清除集群(***高危操作***)
-------------------------------------------------------------------------

* 清除所有的worker信息

  ```
   ./deploy.py kubeadm reset
   rm -rf /etc/cni/net.d/&&rm -rf $HOME/.kube
  ```

* 清除master集群信息

  ```
  kubeadm reset
  rm -rf /etc/cni/net.d/&&rm -rf $HOME/.kube
  ```
